# 台湾版 用語対訳リスト（日本語 → 繁體中文）

使い方サイト・アプリUI・ヘルプ文言すべてで**この表の訳語に統一**すること。
実訳の一次ソースは以下。矛盾したらこの順で優先：

1. **UIボタン/ラベル名** … `Localizable.xcstrings`（アプリ画面表示と一字一句一致させる）
2. **説明文の言い回し** … `Resources/helptext_zhHant.json`（アプリ内ヘルプ。サイト本文はここを流用）
3. **単語ルール** … 本ファイル

---

## 0. 固定ルール（絶対）

| 項目 | ルール |
|------|--------|
| アプリ名 | **ヤメッター → 戒特**（日本語版は「ヤメッター」のまま）。`戒特Premium` |
| スマやめ | 別アプリなので繁体字でも**原文「スマやめ」のまま**（訳さない） |
| ☆ / ランク表記 | ☆記号はそのまま。「ランク」は **等級** |

## 注意（表記ゆれ・要統一）

- 「わかる」は本表で **我懂** に統一。アプリ内 `GJ・わかる` は `GJ・我懂`、設定通知の `グッジョブ・わかる` は現状 `讚・懂了` になっており不一致。サイトでは **讚・我懂** に寄せること（アプリ側は次回合わせて調整予定）。
